# The Whittle School

## Setup

1. **Clone the repo:**

```sh
$ git clone https://github.com/workco/whittle-school.git
$ cd whittle-school
```

2. **Install dependencies**:

```sh
$ yarn
```
  
3. **Start dev server**:

```sh
$ yarn develop
```
  
4. **Build locally and start server**:

```sh
$ yarn start
$ yarn build
```
